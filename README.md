# 项目结构
```bash

├── backend chat-ui后端服务
│   ├── .env 环境变量
│   ├── main.js 入口文件
│   ├── apikey.js deepseek apikey 
├── frontend chat-ui前端
│   ├── src
│     ├── typing
│     ├── App.vue
│     ├── main.ts
│   ├── index.html 入口html文件
│   ├── vite.config.ts vite配置
├── packages 包管理

```

## Setup

Make sure to install dependencies:

```bash

文件根目录下执行install命令
# npm
npm install

```

## Start

```bash

# server

进入backend目录执行命令
npm run dev


# web

进入frontend目录执行命令
npm run dev
```
